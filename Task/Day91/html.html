1.概念不同 href用于在当前文档和引用资源之间确立联系 src用于将资源替换当前元素 2.解析方式不同 href解析时，会并行下载资源且不会停止当前文档处理 src解析时，会暂停当前文档处理

请描述下null和undefined的区别是什么？这两者分别运用在什么场景

null 是“空值”，表示某个对象/资源并不存在；undefined 是指一个属性或变量还未赋值。 当一个变量本该存储一个对象，或一个函数应该返回一个对象，但这个对象却并不存在时，使用 null 来表示这种情况； 当一个属性或变量没有赋值，又或是一个函数没有返回值（如 C++ 里的 void），则它是 undefined；但通常我们不会将一个变量主动地赋为 undefined。

说说你对robots文件的理解，它有什么作用

hosts 文件可以将名称映射到 IP 地址。 在本机上所有对这个名称的访问相当于对它被映射到的 IP 地址的访问。可以说它起到了简易的本地 DNS 的作用。
