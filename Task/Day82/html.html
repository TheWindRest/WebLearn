try { ... throw ... ... } catch (err) { ... } finally { ... } 通过 throw 语句抛出错误；理论上可以抛一切值，但实际上建议只抛 Error 对象； try 块内 throw 的错误会导致停止执行，并将抛出的对象传给 catch 块； 从 ES2017 开始，如果不需要获取抛出的对象，则 catch 块 可以直接写为 catch { ... } catch 块一般用于对错误进行处理； finally 块中的语句不论是否抛出错误，都会执行。
